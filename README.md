Simon Game is a memory-based game that challenges you to repeat a growing sequence of colors and sounds.

üéØ Objective
Repeat the exact pattern of colors generated by the game. With each level, the pattern gets longer by one step.

üß† How to Play
Start the Game

Click the Start button or press any key (depending on your implementation) to begin.

Watch the Pattern

The game will highlight a sequence of buttons in a specific order.

Each button will light up and play a unique sound.

Repeat the Pattern

Click the buttons in the same order they were shown.

If you click the correct sequence, you move to the next level.

Advance Through Levels

Each level adds one more color to the sequence.

The sequence will always start from the beginning but will be one step longer.

Game Over

If you click the wrong color, the game ends.

You'll hear a "wrong" sound and the screen may flash to indicate a mistake.

Press any key (or restart button) to play again.
